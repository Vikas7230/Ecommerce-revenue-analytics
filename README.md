# E-Commerce Revenue & Customer Analytics

## Overview
Analysis of 500K+ retail transactions across 38 countries using Python.

## Key Findings
- Identified top revenue-driving products and seasonal demand spikes
- Built RFM segmentation model classifying customers into 4 behavioural cohorts
- Delivered 5-chart executive dashboard exported to Excel

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Excel (openpyxl)

## Dataset
UCI Online Retail II — via Kaggle
